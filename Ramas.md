# Comandos Ramas I
1.Ver listado de ramas:

&nbsp;
`git  branch`

&nbsp;
2.Crear una rama:

&nbsp;
`git  branch  nombre_rama`

&nbsp;
3.Cambiarnos a una rama:

&nbsp;
`git  checkout  nombre_rama`

&nbsp;
4.Crear una rama y moverse en un paso:

&nbsp;
`git  checkout  -b nombre_rama`

&nbsp;
5.Comparar ramas:

&nbsp;
`git  diff  nombre_rama .. nombre_rama`

&nbsp;
